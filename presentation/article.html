Smart Fallback — No Blind Downgrades

When Claude isn't available, the system matches tier, not just cost. 

# Only GITHUB_TOKEN set — no Claude key:

review_code   (claude / powerful) → Llama-3.1-405B      ← also "powerful"
analyze_repo  (claude / fast)     → gpt-4o-mini          ← also "fast"
plan_tasks    (claude / balanced) → gpt-4o               ← also "balanced"

# With LOCAL_LLM_URL=http://localhost:11434/v1 (local wins over github_models):

review_code   (claude / powerful) → deepseek-coder-v2    ← local, zero data exposure
Blindly routing Opus tasks to gpt-4o-mini would cut quality by 60%+.
Tier-matching keeps quality consistent regardless of which provider is active.
